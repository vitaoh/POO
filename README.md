# Projeto de Programação Orientada a Objetos (POO) em Java

Este projeto foi desenvolvido com o intuito de demonstrar os conceitos e princípios fundamentais da Programação Orientada a Objetos (POO) utilizando a linguagem Java. A POO é um paradigma de programação amplamente utilizado que organiza o código em objetos, facilitando a manutenção, reutilização e escalabilidade do software.
